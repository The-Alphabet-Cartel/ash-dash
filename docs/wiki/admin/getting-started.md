# Getting Started with Ash-Dash

Welcome to Ash-Dash, the crisis detection dashboard for [The Alphabet Cartel](https://discord.gg/alphabetcartel) community. This guide will help you get oriented with the dashboard and its features.

## What is Ash-Dash?

Ash-Dash is part of the Ash ecosystem‚Äîa suite of tools designed to detect and respond to mental health crises within our Discord community. The dashboard provides:

- **Real-time monitoring** of crisis alerts
- **Session management** for tracking interventions
- **User history** for pattern recognition
- **Documentation** and training resources
- **Reporting** and analytics

## The Ash Ecosystem

Ash-Dash works alongside:

| Component | Purpose |
|-----------|---------|
| **Ash-Bot** | Monitors Discord messages |
| **Ash-NLP** | Analyzes messages for crisis indicators |
| **Ash-Dash** | Dashboard for CRT members (you are here) |
| **Ash-Thrash** | Testing suite |

## Dashboard Overview

### Navigation

The sidebar provides access to:

- **Dashboard** - Overview metrics and active sessions
- **Sessions** - All crisis sessions (active and closed)
- **Documentation** - Training materials and guides (this wiki!)
- **Admin** - System settings (admin only)

### Dashboard Page

The main dashboard shows:

- Active crisis sessions requiring attention
- 24-hour metrics (sessions, responses, average response time)
- Crisis trends over time
- CRT activity chart

### Sessions Page

View and manage crisis sessions:

- **Active Sessions** - Requiring CRT attention
- **All Sessions** - Complete history with filtering
- **Session Details** - Full context, notes, and user history

## Your First Session

When a crisis is detected:

1. **Alert appears** on the dashboard
2. Click to **view session details**
3. **Review** the flagged messages and user history
4. **Respond** according to CRT protocols
5. **Document** your actions in session notes
6. **Close** the session when resolved
